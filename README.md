# MLOps Unit 1 Project

## 📌 Project Objective
The objective of this project is to build a simple Machine Learning pipeline using Logistic Regression. It demonstrates data loading, preprocessing, model training, evaluation, and saving the model.

## 📊 Dataset Used
- Titanic Dataset  
- Source: https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv  

## ⚙️ Steps to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/mlops-unit1.git

2. Navigate to the project folder:
   cd mlops-unit1

3. Install dependencies:
   pip install -r requirements.txt

4. Run the script:
   python src/experiment2.py

## 🔁 Reproducibility (Environment Setup)

Follow these steps to recreate the environment:

1. Clone the repository:
   git clone https://github.com/your-username/mlops-unit1.git

2. Navigate to the project folder:
   cd mlops-unit1

3. Create virtual environment (optional but recommended):
   python -m venv venv

4. Activate virtual environment:
   On Windows:
   venv\Scripts\activate

5. Install dependencies:
   pip install -r requirements.txt

6. Run the project:
   python src/experiment2.py
